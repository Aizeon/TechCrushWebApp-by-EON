CI/CD Website Deployment

Build and deploy a web application using a CI/CD pipeline.

Requirements:

* Develop a website using any framework of your choice (e.g., React, Next.js, plain HTML/CSS, etc.).
* Implement a CI/CD pipeline to automate the build and deployment process.
* Deploy the application to a virtual environment using any cloud provider (e.g., Azure, AWS, GCP, etc.).
